| git +                      | Meaning                              |
| -------------------------- | ------------------------------------ |
| `status`                   | Display current status of branch     |
| `log`                      | Show a log of commits                |
| `log --oneline`            | Show a condensed log of commits      |
| `add <path>`               | Stage file(s)                        |
| `add .`                    | Stage all files in working directory |
| `` | Show a log of commits |
