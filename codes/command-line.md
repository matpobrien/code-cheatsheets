# Command Line

## Filepaths

| Command        | Meaning                                                     |
| -------------- | ----------------------------------------------------------- |
| `.`            | Current Working Directory                                   |
| `..`           | Parent Directory                                            |
| `~`            | User Home directory                                         |
| `img/logo.png` | Relative path: resolved relative to current working directy |
| `usr/bin/bash` | Absolute path: resolved relative to the file system root    |
